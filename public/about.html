¿Quiénes somos?


Somos desarrolladores web que se comprometen a trabajar de la mejor manera para hacer entrega de un sitio web funcional.

Organización

Proyecto organizado en diferentes carpetas para los diferentes recursos utilizados.

Funcionalidad

Todas las operaciones de la pagina web son completamente funcionales.

Responsividad web

Esta pagina web puede ser visualizada en cualquier dispositivo con cualquier navegador.

Diseño web agradable

Utilizando HTML5 con diferentes hojas de estilo hemos hecho una pagina web agradable y dinámica.

Vinculación

La pagina web redirigirá al usuario a cualquier parte del sitio de una forma fácil de entender gracias a un menú principal.

Repositorio

El manejo de versiones es vital en el desarrollo web, por eso hemos vinculado esta pagina web a un repositorio en GitHub.
